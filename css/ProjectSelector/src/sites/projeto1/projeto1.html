C O N T E Ú D O S:



HTML5 é a quinta e mais recente versão da linguagem de marcação de hipertexto (HTML), usada para estruturar o conteúdo da web. Ele oferece novas funcionalidades e elementos semânticos para aprimorar a experiência do usuário e a eficiência do desenvolvimento web.

CSS, ou Folhas de Estilo em Cascata, é uma linguagem usada para descrever a apresentação de documentos HTML ou XML. Em termos simples, o CSS define como os elementos de uma página web devem ser exibidos, incluindo cores, fontes, layout e outros aspectos visuais. O CSS trabalha em conjunto com o HTML, que define a estrutura do conteúdo, enquanto o CSS adiciona estilo e apresentação.

O que é a propriedade Display em CSS


Um display (ou mostrador, em português) é um dispositivo para a apresentação de informação, de modo visual e/ou táctil, adquirida, armazenada ou transmitida sob várias formas. Quando a informação de entrada é fornecida como um sinal elétrico, o display é chamado de "display (ou "painel") eletrônico".

Display Características
Block Em CSS, display: block faz com que um elemento HTML seja renderizado como um bloco, ocupando toda a largura disponível do elemento pai e iniciando em uma nova linha. Elementos com display: block por padrão não permitem que outros elementos se alinhem ao seu lado, a menos que propriedades como float sejam usadas. Exemplos de elementos com comportamento padrão block incluem (parágrafos), (divisões) e cabeçalhos como
Inline Um elemento com display: inline é exibido na mesma linha que outros elementos, ocupando apenas o espaço necessário para seu conteúdo (o que é diferente de block, que vai ocupar todo o espaço disponível, mesmo que sobre espaço após o seu conteúdo).
Inline Block O display: inline-block em CSS faz com que um elemento seja tratado como um bloco, mas ainda mantenha o comportamento de um elemento inline, permitindo que vários elementos sejam exibidos lado a lado e que propriedades como largura e altura sejam aplicadas
Flex O display: flex no CSS transforma um elemento em um contêiner flexível, permitindo que seus filhos (itens flexíveis) sejam dispostos e alinhados de maneira flexível, tanto horizontal quanto verticalmente. É uma ferramenta poderosa para criar layouts responsivos e organizar elementos de forma dinâmica.
